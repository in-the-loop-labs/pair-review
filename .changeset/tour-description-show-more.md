Tour-stop descriptions are no longer cut off mid-sentence. The storage cap is raised from 280 to 800 characters and the UI clamps the visible text to ~3 lines, revealing the rest behind a "Show more" toggle on stops that actually overflow. The toggle only appears when needed, and a stop's expanded state is preserved if you scroll past and return to it within the same tour.
