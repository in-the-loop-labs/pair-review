Pair Review

AI-powered GitHub pull request review assistant

Usage

To review a pull request, use one of these commands:

# Review by PR number (from repository directory)
npx pair-review 123

# Review by full GitHub URL  
npx pair-review https://github.com/owner/repo/pull/123

Configuration:

Set your GitHub token in ~/.pair-review/config.json:

{
  "github_token": "your_github_token_here",
  "port": 3000,
  "theme": "light"
}
